# الفصل الخامس عشر: التعامل مع الوقت والتاريخ (DateTime)

## 🕒 الدرس التاسع والخمسون: الوقت المحلي والعالمي (Local / UTC Time)

في هذا الفصل، سنتعلم كيف نجعل برامجنا تدرك الوقت والتاريخ الحاليين. للتعامل مع الوقت في C++، نستخدم Library موروثة من لغة C القديمة تُسمى <ctime>.

### 1. تحذير الأمان في Visual Studio (#pragma warning):

بما أن دوال مكتبة <ctime> قديمة وموروثة من لغة C، فإن الـ Compiler الخاص بـ Visual Studio يعتبرها (Unsafe) ويرفض تشغيل البرنامج ويعطيك Warning صريح ويوقفه عن العمل.

الحل: لتجاوز هذا المنع وإخبار الـ Compiler بأن يتجاهل هذا التحذير، يجب أن نضع هذا السطر في أعلى الكود دائماً:
#pragma warning(disable : 4996)

### 2. كيفية جلب الوقت المحلي (Local Time):

الوقت المحلي هو الوقت الخاص بالمنطقة الجغرافية (Timezone) التي يعمل فيها جهازك (مثلاً: بتوقيت الأردن أو مصر).

نُعرف Variable من نوع خاص يُسمى time_t (وهو مخصص لتخزين الوقت).

نستخدم الدالة time(0) لجلب الوقت الحالي وتخزينه.

نستخدم الدالة ctime(&t) لتحويل هذا الوقت إلى String قابل للطباعة على الشاشة، ونمرر لها الـ Pointer (باستخدام &).

### 3. كيفية جلب التوقيت العالمي (UTC / GMT Time):

التوقيت العالمي (توقيت جرينتش) هو التوقيت القياسي الموحد (لا يتأثر بمكان جهازك).

نجلب الوقت الحالي بنفس الطريقة السابقة.

نستخدم الدالة gmtime(&t) لتحويله إلى توقيت عالمي. هذه الدالة تُرجع Pointer يؤشر على Structure خاص يُسمى tm (سندرسه بالتفصيل في الدرس القادم).

لتحويل هذا الـ Structure إلى String مقروء، نستخدم الدالة asctime().

💻 تطبيق عملي بالكود:

<div dir="ltr">

```cpp
#pragma warning(disable : 4996)
// 🚨 هذا السطر إجباري لتفادي توقف البرنامج في Visual Studio

#include <iostream>
#include <ctime> // Library الخاصة بالوقت والتاريخ
using namespace std;

int main()
{
    // 1. جلب الوقت الحالي للجهاز
    time_t t = time(0);

    // --- طباعة الوقت المحلي (Local Time) ---
    // الدالة ctime تقوم بتحويل الوقت إلى String مقروء
    char* dt = ctime(&t);
    cout << "Local Date and Time is: " << dt << "\n";


    // --- طباعة التوقيت العالمي (UTC / GMT Time) ---
    // الدالة gmtime تُرجع Pointer لـ Structure من نوع tm يحتوي على التوقيت العالمي
    tm* gmtm = gmtime(&t);

    // الدالة asctime تقوم بتحويل هذا الـ Structure إلى String مقروء
    dt = asctime(gmtm);
    cout << "UTC Date and Time is: " << dt << "\n";

    return 0;
}
```

</div>

### 💡 الفائدة الهندسية:

في الأنظمة الكبيرة وقواعد البيانات (مثل حفظ وقت إرسال رسالة أو إنشاء حساب)، من الخطأ الفادح حفظ الـ Local Time لأن المستخدمين قد يكونون في دول مختلفة. الـ Best Practice دائماً هو حفظ الوقت بصيغة الـ UTC، وعند عرضه للمستخدم، يتم تحويله للـ Local Time الخاص بمتصفحه أو جهازه.

## 📅 الدرس الستون: تفكيك الوقت والتاريخ (Datetime Structure)

في الدرس السابق، رأينا كيف تقوم الدالة بإرجاع Structure كامل يحتوي على تفاصيل الوقت. في هذا الدرس، سنتعلم كيف نستخرج كل مكون (Component) على حدة (مثل السنة، الشهر، اليوم، الساعة، الدقيقة، الثانية) لنتمكن من استخدامها في خوارزميات برامجنا بمرونة وحرية تامة.

### 1. الـ Structure المُسمى tm ومعامل السهم ->:

عندما نستخدم الدالة localtime(&t)، فإنها تُرجع لنا Pointer يؤشر على Structure جاهز في C++ يُسمى tm.
بما أننا نتعامل مع Pointer يؤشر على Structure (كما درسنا في فصل المؤشرات)، يجب أن نستخدم معامل السهم -> للوصول إلى الخصائص الداخلية بدلاً من النقطة ..

### 2. التريكات الهندسية في الـ Structure tm:

لغة C++ تقوم بحساب بعض التواريخ بطريقة برمجية قديمة تعتمد على (الصفر) أو تعتمد على سنة مرجعية، لذلك يجب الانتباه لهذه التعديلات عند جلب البيانات:

السنة (tm_year): تُرجع عدد السنوات التي مرت منذ عام 1900.

🚨 الحل: للحصول على السنة الحالية الصحيحة، يجب دائماً أن تضيف 1900 إلى الناتج (مثال: 122 + 1900 = 2022).

الشهر (tm_mon): تُرجع الأشهر بنظام الـ Index من 0 إلى 11 (حيث يناير = 0، وديسمبر = 11).

🚨 الحل: للحصول على رقم الشهر الصحيح، يجب دائماً أن تضيف 1 إلى الناتج.

اليوم وباقي المكونات: تُرجع قيماً صحيحة وجاهزة للاستخدام المباشر.

💻 تطبيق عملي بالكود:

<div dir="ltr">

```cpp

#pragma warning(disable : 4996)
// 🚨 هذا السطر إجباري لتفادي توقف البرنامج في Visual Studio

#include <iostream>
#include <ctime>
using namespace std;

int main()
{
    time_t t = time(0);

    // جلب الوقت المحلي وتخزينه في Pointer من نوع tm Structure
    tm* now = localtime(&t);

    cout << "--- Extracted Datetime Components ---\n";

    // إضافة 1900 للحصول على السنة الصحيحة
    cout << "Year: " << now->tm_year + 1900 << endl;

    // إضافة 1 للحصول على الشهر الصحيح
    cout << "Month: " << now->tm_mon + 1 << endl;

    cout << "Day: " << now->tm_mday << endl;
    cout << "Hour: " << now->tm_hour << endl;
    cout << "Minute: " << now->tm_min << endl;
    cout << "Second: " << now->tm_sec << endl;

    // أيام الأسبوع (من 0 إلى 6، حيث الأحد = 0)
    cout << "Week Day (Days since Sunday): " << now->tm_wday << endl;

    // أيام السنة (كم يوم مضى منذ 1 يناير)
    cout << "Year Day (Days since Jan 1st): " << now->tm_yday << endl;

    // هل التوقيت الصيفي (Daylight Saving Time) مُفعل أم لا؟
    cout << "Is Daylight Savings Time: " << now->tm_isdst << endl;

    return 0;
}

```
